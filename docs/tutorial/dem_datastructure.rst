********************************************************************************
DEM Datastructure
********************************************************************************

This tutorial covers the following steps: 
    - Creating a model from a template
    - Computing contacts
    - Inspecting element geometry
    - Applying transformations
    - Adding interactions
    - Converting meshes to BREP
    - Visualizing everything using the DEM viewer. 

The example code demonstrates this on a simple arch using
:class:`compas_dem.templates.ArchTemplate`.
